# HRMS Lite Frontend

React frontend for HRMS Lite. Provides a clean UI for managing employees and attendance.

## Features
- Add / View Employees
- Mark Attendance
- Filter Attendance by Date
- Dashboard Summary

## Tech Stack
- React (Vite)
- Axios
- CSS
- Deployed on Vercel

## Deployment
- **Backend (Render)**: https://hrms-lite-backend-6ej8.onrender.com/api/
- **Frontend (Vercel)**: https://hrms-lite-frontend-wine.vercel.app

## Setup
```bash
npm install
npm run dev
